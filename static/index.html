About This Demo

                    
This is a simplified version of the Gatenet dashboard
                        running on Heroku. For security and resource
                        constraints, some features are simulated.

                    
For full functionality, install the Gatenet package
                        locally:

                    
pip install gatenet

                    
Then launch the full dashboard:

                    
python -c "from gatenet.dashboard import launch_dashboard; launch_dashboard()"
